# Design System

The visual language for TokenItDown. It inherits the **AnHourTec** visual system; the product carries its
own wordmark while the company identity stays AnHourTec.

<Callout type="info">
  Scope: this page covers **colors and typography**. **Voice:** professional yet approachable, benefit-led,
  no hype words.
</Callout>

## Typography

A system font stack — native, fast, no web-font load:

```css
font-family: ui-sans-serif, system-ui, -apple-system, "Segoe UI", Roboto,
  Helvetica, Arial, sans-serif, "Apple Color Emoji", "Segoe UI Emoji";
```

| Element | Weight | Notes |
|---|---|---|
| H1 | `font-extrabold` | `tracking-tight`, large display sizes |
| H2–H3 | `font-bold` | Section headings |
| Body | `font-normal` | `leading-relaxed` |
| Muted / captions | `font-normal` | Gray 400 / dark-mode muted |

## Implementation notes

- Colors map to Tailwind tokens; the brand blues align with Tailwind's `blue-600/500/400/300`. Define them
  as CSS custom properties / Tailwind theme tokens in `styles/tailwind.css` so light and dark modes share
  one source of truth.
- Prefer the system font stack — don't add a web-font dependency without a deliberate decision.
- Dark mode is first-class: every surface defines both a light and a dark value above.
